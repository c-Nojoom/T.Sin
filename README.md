# 🚀 پروژه T.Sin

یک ابزار قدرتمند، سریع و مدرن برای مدیریت و راه‌اندازی سرویس‌ها روی سیستم‌عامل اوبونتو.

---

## 📋 پیش‌نیازهای سیستم (System Requirements)

برای دریافت بهترین بازدهی و عملکرد بدون نقص، مشخصات زیر پیشنهاد می‌شود:

* **🖥️ سیستم‌عامل:** Ubuntu 24.04 LTS
* **🧠 حافظه رم (RAM):** حداقل 2 گیگابایت (2 GB)
* **💿 پردازنده (CPU):** حداقل 2 هسته (2 Cores)
* **🌍 لوکیشن پیشنهادی:** آلمان (Germany)
    > 💡 **نکته کلیدی:** هر چقدر سرعت و پینگ بین کلاینت (کاربر) تا سرور اصلی کمتر و کیفیت شبکه بالاتر باشد، سرعت و پایداری اتصال به مابقی لوکیشن‌ها نیز به طور چشمگیری افزایش خواهد یافت.

---

## 🛠️ راهنمای نصب و اجرا (Installation)

### ۱. نصب و اجرای سریع
برای نصب خودکار، کافیست دستور زیر را به صورت کامل کپی کرده و در ترمینال اوبونتو خود اجرا کنید:

```bash
sudo bash -c "$(curl -sL "[https://raw.githubusercontent.com/c-Nojoom/T.Sin/main/install.sh?v=$](https://raw.githubusercontent.com/c-Nojoom/T.Sin/main/install.sh?v=$)(date +%s)" | tr -d '\r')"
```
